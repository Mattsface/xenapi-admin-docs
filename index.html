Welcome to Xenapi Admin Docs.

The files and documents stored in this repository exist to help the Xenapi Admin Project create documentation for xapi cloud management systems such as Xenserver, Xen Cloud Project and Kronos. This repository will be used primarily for our own build systems to create the documentation without polluting the official xen-org source tree.

To get the docs project files

$  git clone https://github.com/Xenapi-Admin-Project/xenapi-admin-docs

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Xenapi Admin Guides

We've created specific guides for what will be included in our documentation and how it will be structured. A portion of this section will be used for manpage and other document templates as well as documentation outlining our publishing system (asciidoc/docbook).

Xenapi Admin Build System

We use a custom set of scripts for building our asciidoc source documentation into manpages and other formats. Files associated with this build system will be found here.
